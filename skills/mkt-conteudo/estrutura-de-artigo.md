# Estrutura de artigo de blog

> ⚠️ **Exemplo/placeholder** — adapte ao padrão editorial real do seu blog.

## Esqueleto
1. **Título** (≤ 60 chars): promessa específica com a keyword principal.
2. **Abertura** (2–3 parágrafos): o problema do leitor em 1 frase → por que importa → o que ele leva do artigo.
3. **Corpo**: 3–5 seções H2 (H3 só se necessário). Cada H2 responde UMA pergunta.
   - Exemplo concreto ou número em toda seção.
   - Listas para passos; prosa para contexto.
4. **Conclusão curta** (sem "Concluindo,"): recap em 2 frases + próximo passo.
5. **CTA**: um só, alinhado ao estágio do funil do artigo.

## Padrões
- 1200–1800 palavras para conteúdo pilar; 600–900 para tático.
- Meta description (≤ 155 chars) sempre junto do rascunho.
- Linkar 2–3 artigos internos relacionados (procurar no Notion pautas já publicadas).
- Entregar com "título final + 2 alternativas de título" para o humano escolher.
